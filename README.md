Wanderlust 🌍

Wanderlust is a full-stack web app where users can explore and share hotel listings. The idea behind this project was to build something similar to a travel listing platform where users can add places, view details, and share their experiences.

Features
User signup and login
Add new hotel listings
Edit and delete listings
Upload images
Add reviews and ratings
View listings with location (map)
Basic error handling and flash messages


Tech Stack
Frontend: HTML, CSS, Bootstrap, EJS
Backend: Node.js, Express
Database: MongoDB
Other: Cloudinary, Mapbox/MapTiler, Passport.js


How to Run Locally
Clone the repo
git clone https://github.com/yourusername/wanderlust.git
Go into the folder
cd wanderlust
Install dependencies
npm install

Add a .env file and configure:
CLOUDINARY_KEY=your_key
CLOUDINARY_SECRET=your_secret
MAP_API_KEY=your_key
SESSION_SECRET=your_secret
Start the server
npm start
Open in browser
http://localhost:3000
About the Project

I built this project to practice full-stack development using the MERN-related stack (without React). It helped me understand how authentication, CRUD operations, and APIs work together in a real project.
